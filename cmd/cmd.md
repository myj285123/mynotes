# cmd的常用指令
##### 清屏
```
cls
```

##### 查看所有dos命令
```
help
```

##### 设置环境变量
```
setx "key" "value"
```

##### 查看网络ip
```
ipconfig
```

##### 设置环境变量
* 当前命令行窗口设置
```
set [key] [value]
```
* 永久设置用户的环境变量(加上 **/m** 是给系统设置环境变量)
```
setx [key] [value] [/m]
```

##### 查看目录下的所有文件
```
dir
```

##### 复制
```
copy [origin path] [copy path]
```

##### 移动
```
move [origin path] [move path]
```

##### ping
```
ping [domain/ip]
```

##### 杀进程
```
taskkill [pid]
```
